# 📝 Modern Blog Website

A modern, responsive, and user-friendly blog website built using **HTML, CSS, and JavaScript**. This project features a clean interface, interactive components, dark mode, blog search, category filtering, and a responsive design that works seamlessly across all devices.

---

## 📌 Features

* 🏠 Responsive Home Page
* 📖 Blog Listing & Single Blog View
* 🔍 Search Blogs by Title
* 🏷️ Category Filtering
* 🌙 Dark/Light Mode Toggle
* ❤️ Like Button
* 💬 Comment Section (Local Storage)
* 📧 Newsletter Subscription UI
* 👤 About Page
* 📞 Contact Page
* 🔝 Scroll to Top Button
* 📱 Mobile-Friendly Design
* ✨ Smooth Animations & Modern UI

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Google Fonts
* Font Awesome
